# 🎬 Netflix Data Analysis (EDA) Project

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on Netflix data using Google Colab. The aim is to uncover insights about content distribution, trends over time, and audience preferences based on the available dataset.

---

## 🎯 Objectives

* Understand the structure of Netflix dataset
* Perform data cleaning and preprocessing
* Analyze trends in movies and TV shows
* Identify popular genres and content types
* Visualize insights using graphs and charts

---

## 🛠️ Technologies Used

* **Python**
* **Google Colab**
* **Libraries:**

  * Pandas
  * NumPy
  * Matplotlib
  * Seaborn

---

## 📂 Dataset Description

The dataset includes:

* Title of content
* Type (Movie / TV Show)
* Director and cast
* Country of production
* Release year
* Rating
* Duration
* Genre (listed_in)

---

## 🔍 Steps Performed

### 1. Data Loading

* Imported dataset into Google Colab using Pandas

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Converted date and duration fields
* Standardized categorical values

### 3. Exploratory Data Analysis (EDA)

* Compared number of Movies vs TV Shows
* Analyzed content added over the years
* Identified top genres on Netflix
* Studied country-wise content production
* Examined ratings distribution

### 4. Data Visualization

* Bar charts for content comparison
* Line plots for yearly trends
* Pie charts for distribution analysis
* Heatmaps for correlations

---

## 📊 Key Insights

* Movies dominate the platform compared to TV Shows
* Significant growth in content after 2015
* Drama and International content are highly popular
* USA and India are among top content producers
* Majority of content falls under mature audience ratings

## 🚀 How to Run the Project

### Option 1: Using Google Colab

1. Open Google Colab
2. Upload the `.ipynb` file
3. Upload the dataset
4. Run all cells

### Option 2: Locally

1. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
2. Run using Jupyter Notebook or VS Code
